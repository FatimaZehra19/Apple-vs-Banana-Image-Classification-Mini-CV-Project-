# Apple-vs-Banana-Image-Classification-Mini-CV-Project-
This is my first structured Computer Vision mini project.


In this project, I:
- Loaded image dataset (Apple vs Banana)
- Resized images with aspect ratio preservation
- Applied padding to make images 128x128
- Converted images to grayscale
- Normalized pixel values
- Flattened images into feature vectors
- Created labeled dataset for ML training
